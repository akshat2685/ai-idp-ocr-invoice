# AI-based Intelligent Document Processing (IDP)

## Document Type
Invoice

## Tools Used
- Python
- Tesseract OCR
- OpenCV
- Regex
- JSON

## Approach
1. Uploaded invoice image
2. Preprocessed image for better OCR
3. Extracted text using Tesseract OCR
4. Extracted key fields using regex
5. Converted extracted data into JSON format

## Accuracy Improvements
- NLP-based entity recognition (NER)
- LLM-based structured data extraction
- OCR post-processing and validation

## Assumptions
- English language invoice
- Standard invoice format
